# Production & Marketing Optimization 

## 📊 Project Overview
This project addresses an industrial profit maximization challenge using mathematical modeling. It focuses on finding the optimal production plan under resource constraints (Machine and Labor hours).

## 🛠️ Methodology & Solutions
The project explores two primary optimization approaches:
1.  **Exact Method:** Implementation of the **Simplex Algorithm** to find the absolute optimal solution.
2.  **Approximate Method:** Application of **Heuristic Random Search** to evaluate near-optimal results.
